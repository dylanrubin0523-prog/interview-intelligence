# Claude Project Instructions

Follow `AGENTS.md` as the primary instruction file.

When implementing:

1. Begin by summarizing the issue and listing files expected to change.
2. Inspect existing patterns before writing new abstractions.
3. Make the smallest complete change.
4. Run the repository checks.
5. Produce a concise handoff containing:
   - completed acceptance criteria
   - files changed
   - commands run and results
   - assumptions
   - remaining risks

When reviewing a pull request, do not rewrite the implementation immediately. First produce findings ordered by severity with exact file and line references.
