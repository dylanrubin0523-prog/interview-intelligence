# Phase 1 Product Specification

## Product statement

Interview Intelligence helps college students prepare for recruiting using structured, user-contributed interview experiences and an organized application tracker.

## Initial audience

College students applying to accounting, finance, consulting, wealth-management, and related business internships.

## Phase 1 features

### Authentication and profiles

Users can create an account, sign in, sign out, and maintain basic career preferences.

### Company and position directory

Users can search normalized companies and positions. Duplicate company records should be minimized.

### Interview reports

Authenticated users can submit a structured report containing:

- company and position
- employment type and location
- application period
- outcome
- number and type of rounds
- timeline
- difficulty
- questions or topics reported for each round
- advice
- optional anonymous public display

Reports enter a moderation state before becoming publicly visible.

### Application tracker

Users can create a private application record and move it among:

- Saved
- Planning to apply
- Applied
- Assessment
- Recruiter screen
- First round
- Second round
- Final round
- Offer
- Rejected
- Withdrawn
- Ghosted

### Text interview preparation

A user can select a job or paste a job description and receive practice questions. Questions must distinguish:

- user-reported question or topic
- job-description-derived question
- general practice question

The system must not falsely claim that an inferred question was previously asked.

## Explicitly out of scope for Phase 1

- Realtime voice interviews
- Mobile-native applications
- Subscription billing
- Broad job scraping
- Workplace employee reviews
- News aggregation
- Salary information
- Employer accounts

## Success measures

- A new user can add and update an application in under two minutes.
- A contributor can submit an interview report in under seven minutes.
- Search results clearly separate verified company records from pending records.
- Private application data is never visible to another user.
